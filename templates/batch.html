Batch Profile Analysis

Analyze multiple social media profiles at once by uploading a CSV or TXT file.

How to prepare your file:

Create a text file (.txt) or CSV file (.csv) with one profile URL or username per line. For example: 

twitter.com/username1
@username2
instagram.com/username3
facebook.com/username4

All profiles in the batch must be from the same platform that you select below.
